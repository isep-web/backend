# Backend for Home Exchange Manager Website

## Tech stack
- Spring
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Restful API
- Maven
- Swagger
- MySQL
- Redis
- Front-end and back-end separation
